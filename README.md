# 코드구현 프롬프트

[페르소나]
너는 바닐라 HTML/CSS/JS 프론트엔드 개발자야. 프레임워크·빌드도구 없이 작업해줘.

[기획서]
| 항목 | 내용 |
| :--- | :--- |
| **1) 캐치프레이즈** | "신기루에 흔들리지 않고, 함께 오아시스를 찾아나서요" / "샤워할 땐 온갖 생각을, 행동할 땐 초지일관을." |
| **2) 매력 포인트** | 1. 사색과 실행의 완벽한 밸런스 (아이디어를 현실 코드로 정제)<br>2. 단단함 속에 숨겨진 친근함 (경청과 배려의 커뮤니케이션)<br>3. 초지일관 변함없는 꾸준함 (슬럼프 제어 및 강력한 회복 탄력성) |
| **3) 타겟 방문자** | SSAFY 동기 및 강사님, 잠재적 채용 담당자 |
| **4) 스토리텔링 흐름** | 황량한 사막(사색과 탐색)에서 시작하여 동기들과 함께 단단한 로직을 구축하고, 결국 목표하는 청량한 오아시스(조기 취뽀 및 엔드포인트)에 도달하는 여정 |
| **5) 무드/톤** | 수줍지만 단단한, 사색적인, 청량한 (테크 다크 무드 기반) |
| **6) 메인 컬러 + 폰트** | 사막 오렌지(#f59e0b) & 오아시스 시안 블루(#06b6d4) / Plus Jakarta Sans, Noto Sans KR, JetBrains Mono |

[요청]
기획서 기반으로 1페이지 웹사이트를 만들어줘.

구조: 
1. 상단 헤더 (온라인 상태 바 및 SSAFY 환경 로그 표시)
2. 히어로 섹션 (캐치프레이즈, 소개 문구, 기술 배지 및 스크롤 다운 가이드)
3. 프로필 인터랙티브 콘솔 섹션 (Tab UI를 활용하여 'whoami', 'mbti_details', 'interests' 자바/JSON 코드 스타일 스타일링 및 상세 텍스트 표기, 중앙 정렬된 캐릭터 GIF 포함)
4. 초지일관 성장 타임라인 섹션 (PHASE 01~03 세로형 보더 라인 UI 마일스톤)
5. 핵심 기둥 섹션 (3가지 강점 카드를 그리드 레이아웃으로 배치, 호버 애니메이션 포함)
6. 가상 터미널 방명록 섹션 (Terminal 쉘 GUI 스타일로 제갈량, 제프 베조스, 일론 머스크의 명언 로그 출력)
7. CTA 섹션 (조기 취뽀를 독려하는 강렬한 타이틀과 '함께 취뽀하기' 버튼)
8. 푸터

[인터랙션 및 디테일 추가 요구사항]
- **동적 테마 스위칭 시스템**: 스크롤 깊이(임계값 40%)에 따라 글로벌 루트 변수(--accent, --bg-card, --border-color)가 상단 '사막 오렌지' 테마에서 하단 '푸른 오아시스' 테마로 부드럽게 동적 모핑(색상 전환)되도록 구현해줘. body 클래스 `oasis-active` 제어 및 SVG 언덕 path 색상 변경 포함.
- **풀스크린 ambient 파티클 시스템**: `<canvas>` 엘리먼트를 배경에 깔고, 사막 테마일 때는 오른쪽으로 부드럽게 날리는 작고 고운 황금빛 모래알 입자가, 오아시스 테마로 전환되면 바닥에서 위로 영롱하게 솟구치며 발광(shadowBlur)하는 물방울 입자가 생성·애니메이션되도록 Vanilla JS로 정밀하게 구현해줘.
- **콘솔 탭 전환 구현**: `switchTab()` 함수를 작성하여 탭 클릭 시 `.active` 클래스가 토글되며 해당하는 코드 박스 콘텐츠만 깔끔하게 페이드인(`@keyframes fadeIn`)되도록 처리해줘.
- **커스텀 가상 토스트 로거 및 타이머 액션**: CTA 버튼을 누르면 브라우저 기본 alert 대신 화면 우측 하단에서 탄력적으로 슬라이드 업 되는 커스텀 토스트 알림창(`showToast()`)이 뜨게 하고, 1초(1000ms) 뒤 새 탭으로 깃허브 주소(`https://github.com/77romin`)가 열리도록 비동기 타이머 시스템을 연동해줘.

[형식]
Canvas에서 바로 미리보기 가능하게. 전체 구조 먼저 만들고, 이후 디테일 다듬기. 코드와 스타일, 스크립트가 완벽히 연동되도록 작성해줘.

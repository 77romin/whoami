"샤워할 땐 온갖 생각을, 행동할 땐 초지일관을."

무한한 사색을 견고한 Java 로직으로 구체화하는 백엔드 개발자, 김강민의 단일 페이지 포트폴리오 웹사이트입니다. pi.dev 스타일의 극도로 심플하고 세련된 테크 다크 무드 인터페이스를 바탕으로 설계되었습니다.

📌 Project Overview (프로젝트 개요)

이 프로젝트는 SSAFY 서울 11반 자바 트랙 교육생 김강민의 개발을 대하는 태도, MBTI(INFJ) 성향의 강점, 개인적인 취향 및 비전을 한 페이지 안에 임팩트 있게 녹여낸 반응형 포트폴리오입니다.
가볍고 빠른 정적 웹사이트를 위해 라이브러리나 프레임워크를 전혀 사용하지 않는 순수 HTML, CSS, JavaScript(바닐라 JS)로 빌드되었습니다.

🛠 Tech Stack (기술 스택)

Markup: HTML5 (Semantic Markup)

Styling: CSS3 (Custom Properties, Flexbox, CSS Grid, Transitions/Animations)

Scripting: Vanilla JavaScript (ES6+)

Typography: Plus Jakarta Sans, Noto Sans KR, JetBrains Mono (Google Fonts)

📂 File Structure (디렉토리 구조)

이 프로젝트는 깔끔한 유지보수를 위해 마크업, 스타일, 비즈니스 로직을 완벽히 분리하여 설계되었습니다.

.
├── index.html          # 메인 구조 및 마크업 파일
├── css/
│   └── style.css       # 레이아웃, 애니메이션 및 테마 스타일시트
└── js/
    └── main.js         # 인터랙티브 탭 제어 및 토스트 시스템 스크립트


✨ Key Features (핵심 기능)

1. 🖥️ Interactive Console Box (인터랙티브 콘솔 창)

상단 탭(whoami, mbti_details, interests)을 클릭할 때마다 자바 소스코드 및 JSON 데이터 형식의 유쾌한 코드 로직과 설명이 전환됩니다.

가독성이 뛰어난 개발자 전용 코딩 서체인 JetBrains Mono 서체를 전면 도입하여 디테일을 살렸습니다.

2. 📍 Milestone Timeline (초지일관 성장 타임라인)

김강민 개발자의 학습 진행 흐름(Core Java 입문 ➔ 자료구조/알고리즘 ➔ Web Architecture & Collaboration)을 세밀한 보더 라인 타임라인 UI로 시각화하여 스크롤하는 재미를 더했습니다.

3. 💬 Terminal Guestbook GUI (가상 터미널 방명록)

실제 동기들이나 강사님의 피드백을 가상 터미널 쉘 커맨드 출력 로그 형식으로 시각화하여, 팀 내에서의 평판과 든든한 동료애를 위트 있게 묘사했습니다.

4. 🚀 Custom Toast Logger (커스텀 토스트 알림)

브라우저의 딱딱한 기본 경고창(alert())을 제거하고, 우측 하단에서 슬라이드 업 형태로 탄력적으로 등장하는 모던한 시스템 토스트 모듈을 수동 빌드하여 사용자 경험(UX)을 대폭 개선했습니다.

5. 📱 Fully Responsive Design (완벽한 반응형 지원)

모바일 디바이스부터 데스크톱 4K 환경까지 왜곡 없이 넓고 시원한 여백을 체감할 수 있도록 반응형 브레이크포인트(@media)가 꼼꼼히 설계되었습니다.

🏃‍♂️ How to Run (실행 방법)

별도의 빌드 도구, Node.js 패키지 설치 또는 서버 환경 구성이 전혀 필요하지 않습니다.

본 저장소의 코드를 로컬 디렉토리에 복제합니다.

루트 디렉토리에 있는 index.html 파일을 크롬(Chrome)이나 사파리(Safari) 등 웹 브라우저에서 더블 클릭하여 바로 실행합니다.

또는 VS Code 환경에서 Live Server 익스텐션을 설치하여 로컬 호스트 서버(http://127.0.0.1:5500) 환경에서 편리하게 테스트할 수 있습니다.
